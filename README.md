### 勉強メモ
 - Class
    - メソッド : クラス内での関数を指す. ※ 機能の一かたまり 
      - method(メソッド)の呼び出しにはアロー演算子(->)を使う 
      - アクセス修飾子
        - public    : クラスの外側から呼び出し, 参照, ができる
        - private   : 自分のクラスの内側のみ, 参照, 呼び出しができる 
        - protected : 自分のクラスの内側または自分のクラスを継承したクラスの内側からのみ参照, 呼び出しができる
